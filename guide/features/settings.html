AI+ Application Settings User Manual ​

1. Feature Overview ​

The AI+ application settings allow users to configure basic parameters for AI products, including domain settings and language support, ensuring proper operation and multilingual environment adaptability.

2. Core Features ​

2.1 AI Product Settings ​

  • Description: Configure basic parameters for AI products, such as domains and selectors, with added support for new AI products.
  • Usage:
    1. Open the AI+ application.
    2. Click the settings icon to enter the "AI Product Settings" page.
    3. Enter a valid domain (e.g. http://localhost) in the "Domain" field.
    4. Enter the chart input selector in the "Selector" field.
    5. Click "Save Changes" to apply settings.

2.2 Language Switching ​

  • Description: Add multilingual support for AI products.
  • Usage:
    1. Click the settings icon and select desired language (e.g. Chinese or English).

3. Feature Details ​

3.1 AI Product Settings ​

  • Domain: Input field for AI product access address - ensure validity.
  • Selector: Input field for specifying chart input selector - ensure it matches chart elements in application.
  • Save Changes: Click to save all settings.

4. Notes ​

  1. Domain Validity: Ensure entered domain is valid for proper AI product access.
  2. Selector Matching: Ensure selector matches chart elements in application to prevent functionality issues.
